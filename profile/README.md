# civic-source

**Open tools for public information.**

We build independent civic tech tools that make public information accessible, searchable, and trackable.

## Projects

| Project | Description | Status |
|---------|-------------|--------|
| [us-code-tracker](https://github.com/civic-source/us-code-tracker) | Ingestion engine and static site for tracking changes to the U.S. Code | In development |
| [us-code](https://github.com/civic-source/us-code) | The U.S. Code as Git-versioned Markdown | In development |
