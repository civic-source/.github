# civic-source

Community health files and organization-level configuration for the [civic-source](https://github.com/civic-source) organization.

## What is civic-source?

Independent civic tech tools that make public information accessible, searchable, and trackable.

## Projects

- [us-code-tracker](https://github.com/civic-source/us-code-tracker) — Pipeline and web app for tracking changes to the U.S. Code
- [us-code](https://github.com/civic-source/us-code) — Git-versioned statutory data (CC0 public domain)
- [cap-citation-pipeline](https://github.com/civic-source/cap-citation-pipeline) — Case law citation extraction
